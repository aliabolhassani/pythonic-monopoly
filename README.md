# Unit 6 - Pythonic Monopoly

## Installation Guide

Install the requirements by running the following commands:

```
pip3 install panel
pip3 install plotly
pip3 install pandas
pip3 install hvplot
pip3 install matplotlib
pip3 install os
pip3 install Path
pip3 install dotenv
```
---

## Running

Run the following code on terminal

```
jupyter-notebook
panel serve dashboard.ipynb
```
